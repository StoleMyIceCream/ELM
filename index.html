Visual Explanation of Concepts

1. Input Downsampling

Full Image
(784 pixels)

→

Reduced Features
(e.g., 15-50)

The full 784-pixel image is reduced to a small set of its most informative features. This is the input to the ELM.

2. Temporal Upsampling

1 Input Feature

→

Feature Vector
(e.g., 32 new features)

Each single input feature is nonlinearly expanded into a larger vector of new features, creating the final high-dimensional space.
